Este projeto é um gerador de CPF aleatório desenvolvido em Python, que cria números válidos seguindo as regras oficiais de cálculo dos dígitos verificadores.

O algoritmo funciona gerando os 9 primeiros dígitos de forma aleatória e, em seguida, calcula os dois dígitos finais com base na validação matemática utilizada pela Receita Federal. Isso garante que o CPF gerado seja válido do ponto de vista estrutural.

⚠️ Este código tem fins educacionais e de testes apenas, não devendo ser utilizado para gerar dados reais ou uso indevido.

Funcionalidades:

Geração de CPF válido
Cálculo automático dos dígitos verificadores
Código simples e fácil de entender

Tecnologias:

Python (biblioteca padrão random)
